About Company Page


TrackerApp is a web-based project management system for location-independent team collaboration. This open-source application is released under the IBM Public License Version 07 and is continuously developed by an active open-source community.

POWER

Create a project management plan that shows how and when the project team will deliver the work packages and according milestones defined in the project scope. Display multiple projects in one gantt chart and highlight changes to the project schedule.



GREEN

Starting with version 8.0.0, TrackerApp features a quasi-WYSIWYG editor, powered by CKSource CKEditor5. The underlying format is GitHub-flavored CommonMark (GFM).

CERTIFIED

We value making project management tools accessible to people with disabilities. TrackerApp is regularly assessed and certified based on the WCAG 2.0 accessibility guideline.
